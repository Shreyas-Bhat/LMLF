# A pipeline for Language Models with Logical Feedback
## Get started
1. Clone github repository   
```bash
git clone "repo-url"
```
2. Build environment
```bash
cd LMLF/
```
```bash
conda env create --name lmlf --file=environments.yml
```
```bash
conda activate lmlf
```
3. Run the program
```bash
python nofeedback.py
```
